# Cursor Token 获取助手 - 浏览器插件

专为视力障碍用户设计，支持屏幕阅读器（NVDA / JAWS / VoiceOver），一键从 cursor.com 获取 Access Token。

---

## 📦 安装步骤（Chrome / Edge）

### 第一步：解压插件

将下载的 `cursor-token-extension.zip` 解压到一个**固定的文件夹**，例如：
- Windows：`C:\cursor-token-extension\`
- Mac：`~/cursor-token-extension/`

> ⚠️ 安装后不能删除这个文件夹，否则插件会失效。

### 第二步：打开扩展管理页

**Chrome 浏览器：**
1. 在地址栏输入 `chrome://extensions/` 并按 Enter
2. 或者：菜单 → 更多工具 → 扩展程序

**Edge 浏览器：**
1. 在地址栏输入 `edge://extensions/` 并按 Enter

### 第三步：开启开发者模式

在扩展程序页面右上角，找到"开发者模式"开关，将其**打开**（切换到开启状态）。

### 第四步：加载插件

1. 点击"加载已解压的扩展程序"按钮
2. 在弹出的文件选择框中，选择第一步解压好的文件夹
3. 点击"选择文件夹"

### 第五步：确认安装成功

浏览器工具栏右侧会出现一个蓝色钥匙图标 🔑，说明安装成功。

---

## 🚀 使用方法

1. 先用浏览器访问 [https://cursor.com](https://cursor.com) 并**登录你的账号**
2. 点击工具栏中的蓝色钥匙图标 🔑
3. 在弹出窗口中点击"**一键获取 Access Token**"按钮
4. 稍等片刻，即可看到 Access Token 和 Refresh Token
5. 点击对应的"**复制**"按钮，将 Token 复制到剪贴板

---

## ♿ 无障碍功能说明

- 所有按钮和状态区域均有 ARIA 标签，兼容主流屏幕阅读器
- 操作结果会通过 `aria-live` 区域实时朗读
- 支持纯键盘操作（Tab 切换焦点，Enter/Space 激活按钮）
- 高对比度深色主题，减少视觉疲劳

---

## ❓ 常见问题

**Q: 显示"未检测到登录状态"？**  
A: 请先在浏览器中打开 cursor.com 并完成登录，然后再点击获取按钮。

**Q: 显示"网络请求失败"？**  
A: 请检查网络连接是否正常，或稍后再试。

**Q: Token 多久过期？**  
A: 通常为 60 天，插件会显示剩余天数和过期时间。

---

## 🔐 安全说明

- Token 仅在**本地浏览器**中处理
- 不会将你的 Token 上传至任何第三方服务器（除官方换取 Token 的接口外）
- 建议获取后及时使用，不要长期保存在不安全的地方
